first目录是一个简单的Web应用的例子，它并不是一个eclipse工程。
在其目录下有web.xml文件和HelloServlet.java文件。
该目录可以部署（复制）在tomcat安装目录下的webapps目录下，启动
tomcat后可以访问该Web应用程序，启动浏览器后输入：
http://localhost:8080/first/hello
就可以看到这个Servlet的输出结果了。



second和Servers是eclipse的工程文件，需要导入到eclipse中。然后，可以
运行second目录下的HelloServlet
